# statistical-modeling-r-analysis
Statistical modeling and inference using R (linear models, ANOVA, diagnostics)

## Project Overview
This project contains applied statistical analysis using R, based on a graduate-level statistics course (STA504).

The focus is on building, diagnosing, and interpreting statistical models using real datasets.

## Methods Covered

- Linear Regression
- ANOVA
- Model diagnostics
- Transformation techniques (e.g., Box-Cox)
- Statistical inference
- Power analysis

## Key Skills Demonstrated

- Model building and interpretation
- Assumption checking (linearity, normality, homoscedasticity)
- Residual analysis
- Data-driven decision making

## Files

- `STA504_Final copy.Rmd`: Full analysis with code and explanations
- `STA504_Final copy.pdf`: Final report output

## Tools

- R
- ggplot2
- base R statistical functions
